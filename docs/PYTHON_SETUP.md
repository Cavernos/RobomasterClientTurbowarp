# To setup developpement mode
## For Server Side
```shell
git clone https://github.com/GueuleDeL0up/robomaster-extension-for-scratch.git
python -m venv venv
.\venv\Scripts\activate or source venv/bin/activate on Linux and MacOs
python -m pip install -e .
set ENV=dev
robomaster_cli
```
